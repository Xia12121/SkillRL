Train the prompt, not the weights.

skillrl is a clean, modular Python library for end-to-end optimization of LLM-agent skills — no fine-tuning required. Built on the core algorithm of Microsoft SkillOpt, with an API designed in the spirit of 🤗 TRL.

Why skillrl?

Modern LLM agents are usually improved in one of two ways. Both have hard ceilings. We propose a third path.

🔧

Fine-tune the weights

Expensive, opaque, and often impossible if the model is closed-source.

✍️

Hand-tweak the prompt

Cheap but brittle, ad-hoc, and notoriously hard to reproduce or scale.

✅

Train the skill document

A natural-language markdown skill becomes the trainable state. Both LLMs stay frozen.

SGD on weightsSkillOpt on skill text
Forward pass① Rollout — target agent runs the skill on a batch
Loss computation② Analyze — optimizer LLM diagnoses each failure
Backward pass / gradient③ Patch — optimizer proposes textual edits to the skill
Optimizer step (e.g. Adam)④ Gate — accept the new skill only if it actually improves

How it works

Every training step is a four-stage closed loop. Each stage is a pluggable component you can swap.

STEP 01

Rollout

Run the current skill on a batch of tasks with the frozen target LLM.

STEP 02

Analyze

The frozen optimizer LLM critiques failures and extracts patterns.

STEP 03

Patch

Generate concrete textual edits to the skill document.

STEP 04

Gate

Re-evaluate. Keep the patch only if score improves; otherwise revert.

Quickstart

A complete training loop in under 20 lines. The API mirrors Hugging Face TRL — if you've used PPOTrainer, you already know SkillOptTrainer.

# pip install skillrl
from skillrl import SkillOptConfig, SkillOptTrainer

config = SkillOptConfig(
    num_epochs   = 10,
    batch_size   = 8,
    edit_budget  = 3,
    out_root     = "outputs/run-001",
)

trainer = SkillOptTrainer(
    config      = config,
    target_llm  = my_target_llm,    # any callable: prompt -> str
    optimizer_llm = my_optimizer_llm, # any callable: prompt -> str
    dataset     = my_dataset,
    reward_fn   = my_reward_fn,
)

trainer.train()  # skill evolves; best checkpoint persisted to out_root/
